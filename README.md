## Table of Contents

## Technologies

- **Frontend**: React
  - Node.js version: 18.18
- **Backend**: Python
  - Python version: 3.9.20

  ## Installation

### Backend Setup

    Clone the repository:
    cd backend
    python -m venv venv
    pip install -r requirements.txt
    uvicorn main:app --reload

### Frontend Setup

    Clone the repository:
    cd frontend
    npm install
    npm start
